# Sips & Bites

Premium Ghanaian delicacies — a world-class web experience built with Next.js, Tailwind CSS, and Framer Motion.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- **Cinematic Intro** — Full-screen video overlay with smooth fade-out transition
- **Premium Menu Grid** — Interactive product cards with hover animations
- **AI Order Agent** — Conversational sommelier for recommendations and cart management
- **Glassmorphism Navbar** — Floating header with cart tracking

## Intro Video

Add your intro video to `public/videos/intro.mp4`. A gradient fallback displays if the file is missing.

To replay the intro, clear `sips-bites-intro-seen` from localStorage.

## Tech Stack

- Next.js 15 (App Router)
- Tailwind CSS 4
- Framer Motion
- Lucide React
